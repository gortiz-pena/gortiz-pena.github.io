This is the course website for the Summer 2019 session of ASTR-2600. This class is a first course in Python for astronomy majors at CU Boulder. We will cover Python basics, including JuPyter, iPython, and numpy, as well as data visualization using matplotlib and related libraries.

Pre-Requisites:

There are no formal prerequisites for this course. However, this course will be easier for you if you have experience with high school algebra, geometry, and physics, and have an understanding of scientific notation. It would be extremely helpful if you already have taken or are currently enrolled in calculus.

Why take this class?

The techniques you will learn in this course will enable you to do complex scientific calculations with a powerful programming language. This is a basic requirement for astrophysics research, and for general science, engineering, and information technology. You will have to think logically and linearly, and after this class you will be able teach yourself more extensive computer programming in Python and other programming languages. Thus, this course is for students who want to learn a valuable skill and a way of thinking that will prepare them for astrophysics research or other technical fields.

Goals

This is what students in the class will learn. The broad learning goals will be addressed by the specific competencies that we will be developing together. My hope is that these are appropriate in terms of the time we will have and in terms of the students' prior knowledge. If we find that these goals are either too ambitious or not challenging enough, we can change them. This section will be updated if the course goals change.

Learning Goals

Students who take this class will:

  • Understand the principles of (scientific) programming

  • Learn to think logically and linearly

  • Apply those principles using Python

  • Gain confidence writing complex programs

  • Learn how to learn more Python and other programming languages

Competencies

Students leaving the class can:

  • Navigate a Linux bash terminal

  • Write well-documented, user-friendly code that performs math operations involving the basic data types in Python

  • Distinguish between lists, arrays, and tuples, as well as perform operations on many numbers simultaneously using vector operations and numpy arrays

  • Use logical statements to carry out complicated tasks with programs

  • Write code that fails gracefully using exceptions and error flags

  • Write modular code using functions and classes

  • Debug code, understand the traceback, recognize common Python bugs, and use print statements to know where code is breaking

  • Analyze and visualize numerical data, including common astrophysical data formats and multi-dimensional data sets
